# Nikolan123 Tap

## How do I install these formulae?

`brew install nikolan123/tap/<formula>`

Or `brew tap nikolan123/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "nikolan123/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
